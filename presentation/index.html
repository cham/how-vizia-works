d3

"Data-Driven Documents"

(It's an SVG library)

  1. Vector graphics
  2. Smooth, simple transitions

D3 is not good for:

  1. 'Bitmaps' / photographs
  2. Complex or 3D transitions

We used to use D3 for:

  1. History / Sentiment graphs
  2. Category Bubbles
  3. Globe v1
  4. Sentiment Barometer

But now we use D3 for:

  1. Globe v1
